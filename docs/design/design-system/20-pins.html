20 — Pins

Annotation pins are 30 × 30 teardrop tags rotated -45° (border-radius: 50% 50% 50% 0) so the tip lands on the anchor point. Glass tag aesthetic: dark translucent bg, accent border tied to the annotation's colorIndex (0-15 palette), inner numeric label counter-rotated 45° to read horizontally. opacity: 0.55 at rest so the underlying mockup reads through; full opacity on hover / active. 16-colour palette shared between pins, rail badges, and avatars.
