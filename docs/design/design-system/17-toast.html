17 — Toast

Toast notification system (src/components/Toast/Toast.tsx, useToast.ts). Bottom-center fixed container holds a stack of glass pills. Slide-in from below with spring easing; auto-dismiss after a default duration; out-fade on remove. Pills use the glass-surface standard — same recipe as the rail, toolbar, dialog scrim, and every other floating overlay.
