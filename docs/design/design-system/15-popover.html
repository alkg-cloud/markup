15 — Popover

The single popover primitive (src/lib/popover/usePopover.ts). Wraps the native HTML popover API (popover="auto" + popovertarget) plus positionPopover which anchors to the trigger via getBoundingClientRect, flips above when there's no room below, and clamps to the viewport with a 4 px gutter. Paints in the browser top-layer — escapes every overflow ancestor and stacking context. Browser owns light-dismiss, ESC, and the single-popover-auto invariant.
