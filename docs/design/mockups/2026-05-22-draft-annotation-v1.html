Inline Draft Annotation idea · v1

The "Add comment" modal disappears. Composition lives in the annotations rail as an inline DraftCard pinned to the top. Any click on the canvas adds a pin to the active draft — no marking-mode toggle. Three terminal actions: Cancel, Draft, Send. Content (text + pins) survives reloads via localStorage. See 2026-05-22-draft-annotation-spec.md for the full surface.

F · Behavior summary
  • Entry. User clicks the rail's + New annotation button (or presses N with no input focused). Rail force-pins; DraftCard mounts at top; textarea autofocuses.
  • Canvas click while drafting. Cursor becomes crosshair. Click anywhere → new pin appended (+ pulsing). Click a draft pin → remove it. Click a published pin → focus that card.
  • Draft. ⌘ + S or button → flush to localStorage (key: markup:draft:<mockupId>:<userId>). Auto-save debounces 800 ms on body / pins change.
  • Send. ⌘ + ↵ or button (enabled when body ≥ 1 char and ≤ 10 000 chars; pins optional, 0 – 20). POST → optimistic insert; localStorage cleared; rail returns to user's prior pin state.
  • Cancel. Esc or button. Empty draft → silent discard. Non-empty draft → AlertDialog confirms. Confirm clears localStorage; cancel-of-cancel returns to active.
  • Restore. Viewer mount reads localStorage; if a draft exists for (mockupId, userId), mount DraftCard prefilled. Stale drafts (lastSavedAt > 30 days) discarded silently.
  • Max pins. Anchors capped at 20 (API contract). The 21st click is consumed; toast "Maximum 20 pins per annotation"; pin count row reads 20 pins · max reached (warning hue).
  • Intent type — removed. The chip row that used to live above the textarea is gone. Annotation.intentType is dropped from the schema, the agent-loop /intent endpoint, the /agent/context aggregator, the docs in docs/agent-loop/, and all consumer pages. The DraftCard composition is leaner because of it. (See §10 of the spec.)
  • Reduced motion. Pulse on draft pins is suppressed under prefers-reduced-motion: reduce (via the global motion-token overrides at :root). DraftCard accent-dot animation also collapses.
